# 影院商品管理中台 - 项目规则

## 项目概述
影院商品管理中台，采用前后端分离架构，支持 B端管理后台和 C端用户应用。

## 技术栈分层
- **B端（管理后台）**: React + Ant Design + TypeScript
- **C端（用户端/小程序）**: Taro 框架（多端统一）
- **后端**: Spring Boot + Supabase
