## 概述

本文档描述了 5 个关键页面的 UI 布局和视觉设计，作为新 Taro 实现的参考基准。新实现必须达到 **≥95%** 视觉一致性。

## UI 一致性检查清单

新实现时必须验证以下要点：

### 颜色系统
- [ ] 主色调 `#1989fa` 使用正确
- [ ] 强调色 `#FF6B35` (价格) 使用正确
- [ ] 成功色 `#07C160` 使用正确
- [ ] 警告色 `#FF976A` 使用正确
- [ ] 危险色 `#EE0A24` 使用正确
- [ ] 文字颜色 `#323233` / `#646566` / `#969799` 使用正确
- [ ] 背景色 `#F7F8FA` / `#FFFFFF` 使用正确
- [ ] 边框色 `#EBEDF0` 使用正确

### 字体系统
- [ ] 基础字号 `28rpx` 为默认字体大小
- [ ] 标题使用 `36rpx` 粗体
- [ ] 价格使用 `32rpx` 或 `48rpx` 粗体
- [ ] 辅助文字使用 `24rpx`

### 间距系统
- [ ] 页面左右边距统一 `32rpx`
- [ ] 卡片间距统一 `24rpx`
- [ ] 组件内边距统一 `32rpx`
- [ ] 小间距使用 `16rpx`

### 组件样式
- [ ] 圆角大小统一（按钮 `28rpx`, 卡片 `16rpx`, 图片 `8rpx`）
- [ ] 按钮高度统一 `56rpx` 或 `88rpx`
- [ ] 阴影样式统一 `0 4rpx 12rpx rgba(0, 0, 0, 0.08)`

### 交互效果
- [ ] 按钮有 hover 态（透明度 0.8）
- [ ] 页面切换有平滑过渡动画
- [ ] 列表滚动流畅（FPS ≥ 50）
- [ ] 图片加载有占位符和渐显效果
