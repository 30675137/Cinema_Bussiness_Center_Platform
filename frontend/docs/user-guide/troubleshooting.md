# 常见问题解答

## 登录相关

### Q1: 忘记登录密码怎么办？

**问题描述**: 无法记起系统登录密码

**解决方案**:
1. 在登录页面点击"忘记密码"链接
2. 输入注册时使用的邮箱或手机号
3. 查收邮箱或短信中的重置密码链接
4. 点击链接进入密码重置页面
5. 设置新密码并确认
6. 使用新密码重新登录

**注意事项**:
- 重置链接24小时内有效
- 新密码要求包含字母、数字和特殊字符
- 建议定期更换密码确保安全

### Q2: 登录时提示验证码错误？

**问题描述**: 输入的验证码总是显示错误

**可能原因**:
- 验证码已过期（通常5分钟有效期）
- 验证码输入时区分大小写
- 浏览器缓存问题

**解决方法**:
1. 刷新页面获取新的验证码
2. 确认输入的验证码大小写正确
3. 清除浏览器缓存后重新尝试
4. 如果仍无法解决，联系系统管理员

### Q3: 登录后页面显示空白？

**问题描述**: 成功登录后显示空白页面或无法加载内容

**解决步骤**:
1. **检查浏览器兼容性**:
   - 确认使用支持的浏览器版本
   - 推荐使用Chrome 80+或Firefox 75+

2. **清除浏览器缓存**:
   - 清除浏览器缓存和Cookie
   - 重新登录系统

3. **检查网络连接**:
   - 确认网络连接正常
   - 尝试访问其他网站测试网络

4. **禁用浏览器插件**:
   - 暂时禁用广告拦截等插件
   - 重新加载页面

5. **检查控制台错误**:
   - 按F12打开开发者工具
   - 查看Console标签中的错误信息
   - 将错误信息反馈给技术支持

## 商品管理

### Q4: 创建商品时保存失败？

**问题描述**: 填写完商品信息后点击保存无反应或提示保存失败

**排查步骤**:

1. **检查必填字段**:
   - 确认所有带红色星号的字段已填写
   - 检查字段格式是否正确
   - 验证数字字段是否为有效数字

2. **检查图片上传**:
   - 确认至少上传了一张主图
   - 检查图片格式是否为jpg/png
   - 验证图片大小是否超过限制

3. **检查分类选择**:
   - 确认已选择有效的商品分类
   - 检查分类是否启用状态

4. **网络连接检查**:
   - 检查网络连接是否稳定
   - 尝试刷新页面重新保存

5. **浏览器问题**:
   - 尝试更换浏览器
   - 清除浏览器缓存

**常见错误及解决**:
- "商品名称已存在": 更换商品名称
- "价格格式错误": 确认价格为数字格式
- "分类不存在": 重新选择有效分类

### Q5: 商品图片上传失败？

**问题描述**: 上传商品图片时失败或进度条卡住

**解决方案**:

1. **检查图片格式**:
   - 支持格式：JPG、PNG、GIF
   - 不支持格式：BMP、TIFF、PSD

2. **检查图片大小**:
   - 单张图片不超过5MB
   - 建议大小：100KB-2MB

3. **检查图片尺寸**:
   - 最小尺寸：300×300像素
   - 推荐尺寸：800×800像素

4. **检查文件名**:
   - 避免使用中文字符
   - 避免使用特殊符号
   - 建议使用英文字母和数字

5. **网络问题**:
   - 检查网络连接稳定性
   - 尝试压缩图片后重新上传
   - 分批次上传多张图片

### Q6: 批量导入商品失败？

**问题描述**: 使用Excel模板导入商品时失败或部分失败

**解决方法**:

1. **检查文件格式**:
   - 确认文件为.xlsx格式
   - 检查文件是否损坏
   - 使用系统提供的最新模板

2. **检查数据格式**:
   - 确保日期格式正确（YYYY-MM-DD）
   - 价格字段使用数字格式
   - 文本字段不包含特殊符号

3. **检查必填字段**:
   - 商品名称不能为空
   - 分类ID必须存在
   - SKU不能重复

4. **检查数据长度**:
   - 商品名称不超过100字符
   - 描述不超过1000字符
   - 其他字段按模板要求

5. **分批导入**:
   - 大量数据建议分批导入
   - 单次建议不超过1000条记录

**错误码对照表**:
| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 必填字段为空 | 必填字段未填写 | 补充必填字段 |
| 分类不存在 | 分类ID无效 | 选择有效分类 |
| SKU重复 | 商品SKU已存在 | 修改SKU |
| 价格格式错误 | 价格非数字格式 | 使用数字格式 |

### Q7: 商品搜索不到结果？

**问题描述**: 使用搜索功能找不到已知存在的商品

**排查方法**:

1. **检查搜索条件**:
   - 确认搜索关键词正确
   - 检查筛选条件是否过严
   - 尝试使用更宽泛的关键词

2. **检查商品状态**:
   - 确认商品状态为"已发布"
   - 草稿状态商品不会在搜索结果中显示

3. **检查分类权限**:
   - 确认有权限查看该分类商品
   - 检查分类是否被禁用

4. **检查搜索范围**:
   - 确认搜索范围包括商品名称、描述等
   - 尝试在不同字段中搜索

5. **索引更新**:
   - 新创建商品可能需要几分钟才能被搜索到
   - 联系管理员检查搜索索引状态

## 系统性能

### Q8: 系统运行缓慢？

**问题描述**: 页面加载缓慢，操作响应迟钝

**优化建议**:

1. **浏览器优化**:
   - 清除浏览器缓存
   - 关闭不必要的标签页
   - 禁用占用资源的浏览器插件

2. **网络优化**:
   - 检查网络连接速度
   - 使用有线网络连接
   - 避免网络高峰期使用

3. **数据量优化**:
   - 减少单页显示数据量
   - 使用筛选条件缩小数据范围
   - 避免同时打开多个大数据页面

4. **系统维护**:
   - 定期重启浏览器
   - 更新浏览器到最新版本
   - 定期清理系统垃圾文件

### Q9: 文件上传/下载缓慢？

**问题描述**: 上传或下载文件速度很慢

**解决方法**:

1. **检查文件大小**:
   - 大文件上传需要更长时间
   - 考虑压缩文件后上传

2. **网络带宽**:
   - 测试网络上传/下载速度
   - 联系网络管理员检查带宽

3. **服务器负载**:
   - 避免高峰期进行大文件操作
   - 分批次上传多个文件

4. **使用工具**:
   - 使用专业的FTP工具
   - 考虑使用断点续传功能

## 权限相关

### Q10: 某些功能无法使用？

**问题描述**: 系统中某些按钮或菜单显示灰色，无法点击

**原因分析**:
- 当前用户权限不足
- 功能模块未启用
- 系统维护状态

**解决步骤**:

1. **检查用户权限**:
   - 联系系统管理员确认权限设置
   - 申请相应的功能权限
   - 确认账号状态正常

2. **检查模块状态**:
   - 确认功能模块已启用
   - 检查系统维护公告
   - 联系技术支持确认

3. **重新登录**:
   - 退出系统重新登录
   - 确认权限已生效

### Q11: 无法看到某些数据？

**问题描述**: 在列表中看不到某些商品或数据

**可能原因**:
- 数据权限限制
- 筛选条件过滤
- 数据状态问题

**解决方法**:

1. **检查数据权限**:
   - 确认有权限访问该数据
   - 检查数据所属部门权限
   - 联系管理员调整权限

2. **检查筛选条件**:
   - 清除所有筛选条件
   - 重置页面默认设置
   - 尝试不同的筛选组合

3. **检查数据状态**:
   - 确认数据未被删除
   - 检查数据是否已归档
   - 验证数据状态是否有效

## 数据异常

### Q12: 商品库存数量异常？

**问题描述**: 显示的库存数量与实际不符

**处理流程**:

1. **确认问题**:
   - 核对实际库存数量
   - 检查最近的出入库记录
   - 查看库存变更日志

2. **查找原因**:
   - 检查是否有未同步的订单
   - 确认是否有库存调整记录
   - 查看系统异常日志

3. **调整库存**:
   - 使用库存调整功能
   - 填写调整原因和数量
   - 提交调整申请

4. **预防措施**:
   - 定期进行库存盘点
   - 设置库存预警
   - 规范出入库流程

### Q13: 商品价格显示错误？

**问题描述**: 商品价格显示不正确或计算异常

**排查步骤**:

1. **检查基础价格**:
   - 确认商品基础价格设置
   - 检查价格是否包含税
   - 验证价格格式

2. **检查促销设置**:
   - 查看是否有促销活动
   - 确认促销价格和期限
   - 检查促销规则配置

3. **检查渠道价格**:
   - 确认当前查看渠道
   - 检查渠道特定价格设置
   - 验证价格覆写规则

4. **检查缓存**:
   - 清除浏览器缓存
   - 刷新页面重新加载
   - 联系技术支持检查系统缓存

## 系统报错

### Q14: 遇到"500内部服务器错误"？

**问题描述**: 操作时弹出500错误提示

**含义**: 服务器内部出现错误，通常是程序异常

**解决方法**:

1. **刷新页面**:
   - 刷新当前页面
   - 重新进行操作
   - 如果持续出现，继续下一步

2. **记录详细信息**:
   - 截图保存错误信息
   - 记录操作步骤
   - 记录发生时间

3. **联系技术支持**:
   - 提供错误截图
   - 说明操作过程
   - 提供发生时间

4. **临时处理**:
   - 尝试其他浏览器
   - 稍后再次尝试
   - 使用其他功能替代

### Q15: 遇到"403权限不足"错误？

**问题描述**: 访问某些页面时提示403权限不足

**解决方法**:

1. **确认操作权限**:
   - 检查当前用户角色
   - 确认是否有该功能权限
   - 联系管理员申请权限

2. **检查会话状态**:
   - 确认登录状态正常
   - 重新登录系统
   - 检查账号是否被锁定

3. **路径正确性**:
   - 确认访问路径正确
   - 使用系统导航菜单
   - 避免直接输入URL

```javascript
console.log(navigator.userAgent);

// 查看当前URL

// 查看页面加载时间
console.log(performance.timing);

// 查看本地存储

```
