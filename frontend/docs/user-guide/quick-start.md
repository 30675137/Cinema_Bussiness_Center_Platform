# 快速入门指南

## 系统概述

商品管理中台前端系统是一个现代化的商品管理平台，提供完整的商品生命周期管理功能，包括商品创建、编辑、分类管理、库存管理等核心功能。

## 登录系统

### 1. 访问系统

打开浏览器，输入系统地址：`https://your-domain.com`

### 2. 用户登录

在登录页面输入以下信息：

- **用户名**: 您的系统用户名
- **密码**: 您的系统密码
- **验证码**: 根据显示的验证码输入

```mermaid
graph LR
    A[打开登录页面] --> B[输入用户名密码]
    B --> C[输入验证码]
    C --> D[点击登录按钮]
    D --> E[进入系统首页]
```

### 3. 首次登录设置

首次登录成功后，建议进行以下设置：

1. **修改初始密码**
   - 点击右上角用户头像
   - 选择"个人设置"
   - 在安全设置中修改密码

2. **完善个人信息**
   - 填写真实姓名
   - 设置联系方式
   - 选择时区偏好

## 基本操作指南

### 1. 导航操作

#### 使用左侧菜单

- 点击菜单项进入对应功能模块
- 点击菜单前的展开图标查看子菜单
- 支持菜单搜索和收藏功能

#### 面包屑导航

主内容区顶部显示面包屑导航，显示当前位置：

```
首页 > 商品管理 > 商品列表
```

点击面包屑中的任意节点可以快速返回对应层级。

### 2. 数据查看

#### 列表查看

大部分数据以表格形式展示，支持以下操作：

- **排序**: 点击列标题进行升序/降序排序
- **筛选**: 使用顶部的筛选条件过滤数据
- **搜索**: 在搜索框中输入关键词快速查找
- **分页**: 使用底部分页控件浏览更多数据

#### 详情查看

点击列表中的"查看"按钮或在数据行上双击，可以查看详细信息。

### 3. 数据操作

#### 新增数据

1. 点击页面右上角的"新增"按钮
2. 填写必要的信息字段
3. 点击"保存"完成创建

#### 编辑数据

1. 在列表中找到要编辑的数据
2. 点击"编辑"按钮
3. 修改相关信息
4. 点击"保存"确认修改

#### 删除数据

1. 选择要删除的数据（可多选）
2. 点击"删除"按钮
3. 在确认对话框中确认删除操作

### 4. 批量操作

系统支持批量处理多条数据：

1. 在列表中勾选要操作的数据行
2. 选择批量操作类型（删除、修改状态等）
3. 根据提示完成操作

## 常用快捷键

为了提高操作效率，系统支持以下快捷键：

| 快捷键       | 功能说明 | 适用页面   |
| ------------ | -------- | ---------- |
| Ctrl + N     | 新建商品 | 商品列表页 |
| Ctrl + F     | 搜索     | 列表页面   |
| Ctrl + S     | 保存     | 编辑表单页 |
| Esc          | 取消操作 | 弹窗、表单 |
| F5           | 刷新页面 | 所有页面   |
| Ctrl + Enter | 确认提交 | 表单页面   |

## 个性化设置

### 1. 界面主题

系统支持明暗主题切换：

1. 点击右上角用户头像
2. 选择"界面设置"
3. 在主题选择中选择"浅色"或"深色"

### 2. 语言设置

系统支持多语言界面：

1. 在界面设置中找到"语言设置"
2. 选择所需语言（简体中文、繁体中文、英文）

### 3. 表格设置

可以自定义表格显示：

- **列显示/隐藏**: 选择要显示的列
- **列宽调整**: 拖拽列边框调整宽度
- **每页显示数**: 设置每页显示的记录数

## 注意事项

1. **数据安全**: 不要在公共设备上保存登录信息
2. **定期备份**: 重要数据建议定期导出备份
3. **权限管理**: 根据工作职责申请相应的系统权限
4. **浏览器兼容**: 推荐使用Chrome、Firefox、Safari等现代浏览器

## 系统要求

- **浏览器**: Chrome 80+、Firefox 75+、Safari 13+、Edge 80+
- **屏幕分辨率**: 最低1366×768，推荐1920×1080
- **网络带宽**: 建议不低于1Mbps
- **操作系统**: Windows 10+、macOS 10.14+、Linux主流发行版
